# ADE20K Texture Miner Site

Standalone GitHub Pages site for ADE20K texture awareness/ranking review.

## Local structure
- `docs/review/` - static gallery (ADE20K-only)
- `docs/method/` - research note, ranking evidence, and visual diagnostics
- `docs/index.html` - landing page

GitHub Pages deploys from `docs/` via `.github/workflows/pages.yml`.
